# **App Name**: Elysian Threads

## Core Features:

- Product Browsing: Browse women's fashion items with filtering by category, color, size, fabric, and price.
- Product Details: View detailed product information, including high-resolution images, descriptions, fabric details, and related products.
- Shopping Cart: Add items to a shopping cart and proceed to a simplified checkout process with order placement and confirmation.
- User Authentication: User authentication via Email/Password and Google Sign-in using Firebase Auth.
- Firestore Database: Products, categories, users, orders, and blog posts stored in Firestore.
- Image Storage: Store and serve images of products using Firebase Storage.
- AI Product Recommendations: The AI tool generates personalized product recommendations based on browsing history.

## Style Guidelines:

- Primary color: Light gray (#FAFAFA) to create a minimalist, clean background, inspired by Apple's design philosophy.
- Background color: Off-white (#F2F2F2) to maintain a soft, luxurious feel.
- Accent color: Champagne gold (#E2D7BE) to add subtle highlights and a sense of premium elegance.
- Font: '-apple-system, BlinkMacSystemFont, Helvetica Neue, Arial, sans-serif' (San Francisco) to maintain Apple's signature typography style. Note: currently only Google Fonts are supported.
- Utilize whitespace extensively to ensure the design feels open, airy, and premium.
- Incorporate subtle fade-in and slide-up animations for page transitions and product displays.
- Use minimalist icons for navigation and UI elements to enhance clarity and sophistication.
